# Contributing

## Tooling

- Use the Node version in `.nvmrc`
- Use Yarn (modern) via Corepack:
  - `corepack enable`
  - `yarn --version` should match the repo `packageManager` field

## Repo layout

- `vault/` = Stacks Proof Vault dApp (Next.js)
- `escrow/` = Stacks Micro-Escrow dApp (Next.js)

## Commit discipline

Every task should end with:

- working state
- commit
- push
